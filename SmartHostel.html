Smart Hostel Management System

Developed a management system as part of a 4-member team using Angular. The Smart Hostel Management System is an innovative application designed to streamline hostel operations.

This project integrates features such as real-time occupancy tracking, maintenance requests, and communication channels, making it more efficient and convenient for hostel residents and management alike. The system's user-friendly interface ensures that both the students and hostel administration can easily manage tasks.

Technologies Used: Angular, TypeScript, Bootstrap, graphQL
